1.3 Your First API Call ‚Äî One Turn

The simplest interaction: send a message, get a response.

OpenAI Chat Completions 
POST /v1/chat/completions

{
  "model": "gpt-4o",
  "messages": [
    {
      "role": "system",
      "content": "You are a helpful assistant."
    },
    {
      "role": "user",
      "content": "What is DNS?"
    }
  ]
}
Key: messages array with role + content
Anthropic Messages API 
POST /v1/messages

{
  "model": "claude-sonnet-4-5",
  "max_tokens": 1024,
  "system": "You are a helpful assistant.",
  "messages": [
    {
      "role": "user",
      "content": "What is DNS?"
    }
  ]
}
Key: system is top-level, not in messages. max_tokens required.

Key Differences

System prompt OpenAI Inside messages[] as "role": "system" Anthropic Top-level "system" field
Max tokens OpenAI Optional (defaults vary) Anthropic Required
Endpoint OpenAI /v1/chat/completions Anthropic /v1/messages

1.4 Reading the Response

OpenAI Chat Completions Response 
{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "DNS stands for..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 25,
    "completion_tokens": 150,
    "total_tokens": 175
  }
}
Access: response.choices[0].message.content
Anthropic Messages Response 
{
  "id": "msg_abc123",
  "type": "message",
  "role": "assistant",
  "model": "claude-sonnet-4-5",
  "content": [
    {
      "type": "text",
      "text": "DNS stands for..."
    }
  ],
  "stop_reason": "end_turn",
  "usage": {
    "input_tokens": 25,
    "output_tokens": 150
  }
}
Access: response.content[0].text

Structural Differences

Response text OAI choices[0].message.content (string) Ant content[0].text (content blocks array)
Multiple outputs OAI n param for parallel generations Ant Not supported (one response per call)
Content format OAI Plain string Ant Array of typed content blocks
