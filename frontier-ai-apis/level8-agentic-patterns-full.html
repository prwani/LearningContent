8.2 Pattern: Skill-Driven Workflow

Side-by-side examples showing how skills change the agentic loop.

OpenAI Responses API with Skills 
POST /v1/responses
{
  "model": "gpt-5.2",
  "input": "Generate Q1 financial report",
  "tools": [
    {"type": "skill",
     "skill": {"name": "report-writer"}},
    {"type": "shell",
     "environment":
       {"type": "container_auto"}},
    {"type": "web_search"}
  ],
  "store": true
}

// Agent:
// 1. Loads report-writer skill
// 2. Follows SKILL.md instructions
// 3. Uses shell to process data
// 4. Searches web for benchmarks
// 5. Writes report to /mnt/data/
Anthropic Messages API with Skills 
POST /v1/messages
{
  "model": "claude-sonnet-4-5",
  "max_tokens": 8192,
  "system": "[Skill: report-writer loaded]\n\n# Report Writer Workflow\n1. Gather data from provided sources\n2. Calculate key metrics\n3. Compare with industry benchmarks\n4. Generate executive summary\n5. Format as structured report",
  "messages": [
    {"role": "user",
     "content": "Generate Q1 financial report from the attached data"}
  ],
  "tools": [
    {"name": "calculate",
     "description": "Run calculations",
     "input_schema": {
       "type": "object",
       "properties": {
         "expression": {"type": "string"}
       }}},
    {"name": "search_benchmarks",
     "description": "Find industry data",
     "input_schema": {
       "type": "object",
       "properties": {
         "query": {"type": "string"}
       }}}
  ]
}

// Agent loop:
// 1. Skill instructions in system prompt
// 2. Model follows workflow steps
// 3. Calls tools as needed
// 4. Returns structured report

8.3 Skills vs Tools vs Plugins ‚Äî When to Use What

Concept What it is When to use
Skills Reusable instructions (SKILL.md) Repeatable workflows, domain SOPs, templates
Tools Functions the model can call Real-time data, external actions, computation
Plugins Bundles of skills + commands + connectors Role-based setups (sales, legal, data)
MCP Protocol for external service connections Third-party integrations (Slack, GitHub, DBs)
