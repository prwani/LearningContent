Skills ‚Äî Key Points

Standard Both Open standard from agentskills.io ‚Äî same SKILL.md format works across providers
Loading Progressive disclosure ‚Äî only metadata indexed on startup, full content loaded on demand
Activation OAI Via tools array in API Ant Auto-discovered from .claude/skills/
Best for SOPs, templates, domain expertise, repeatable workflows ‚Äî replaces "mega system prompts"

7.2 Shell, Plugins & Compaction

OpenAI ‚Äî Shell Tool

Full Linux container (Debian 12) with Python, Node.js, Java, Go. Model can install packages, run scripts, write artifacts.

"tools": [{
  "type": "shell",
  "environment": {
    "type": "container_auto"
  }
}]

// Container includes:
// Python 3.11, Node 22, Java 17
// Go 1.23, Ruby 3.1, PHP 8.2
// Persistent /mnt/data for artifacts
// Network access with allowlists
Ephemeral: container_auto ‚Äî fresh per request
Reusable: container_reference ‚Äî persist state across calls

Anthropic ‚Äî Knowledge Work Plugins

Role-based bundles of skills, MCP connectors, and slash commands. 11 open-source plugins. Note: Plugins are an SDK-level concept ‚Äî not directly in the Messages API.

# Plugin structure:
plugin-name/
  .claude-plugin/plugin.json  # Manifest
  .mcp.json        # Tool connections
  commands/        # Slash commands
  skills/          # Domain knowledge

# Install and use:
claude plugin install sales
# Then: /sales:call-prep
#        /sales:pipeline-review

# Plugins bundle:
# - Skills (auto-triggered)
# - Commands (explicit actions)
# - Connectors (MCP servers)
11 plugins: productivity, sales, support, PM, marketing, legal, finance, data, search, bio, mgmt

Context Management for Long-Running Agents

Long-running agents hit context limits. Both providers now offer solutions to keep agents working across extended sessions.

OpenAI ‚Äî Server-Side Compaction

  • Auto: Triggers when context nears limit ‚Äî no code change
  • Manual: /responses/compact endpoint
  • Preserves reasoning continuity mid-stream
  • Works with previous_response_id for stateful sessions

Anthropic ‚Äî Context Editing + Compaction

  • Compaction: Auto-summarization near context limit
  • Context editing: Clear old tool results, thinking blocks
  • Prompt caching: Reuse frequent prefixes (up to 1hr)
  • Memory tool: Persistent storage across sessions
