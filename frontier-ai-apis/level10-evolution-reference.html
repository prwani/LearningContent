10.1 API Evolution Timeline

OpenAI

2023
Chat Completions API launched. Function calling added.
2024
Parallel tool calls. JSON mode & structured outputs. Assistants API (threads + tools).
2025
Responses API launched. Built-in tools (web, code, files). MCP integration. Assistants API deprecated. Agent Skills adopted. Computer use (preview).
2026
Responses API becomes primary. Open Responses specification with partners. Shell tool + Skills + Compaction.

Anthropic

2023
Messages API launched. Basic text completions.
2024
Tool use added. Vision/image support. Computer use (beta).
2025
Advanced tool use. Tool search & programmatic calling. MCP native support. Agent Skills (open standard).
2026
Opus 4.6 with 1M context. Knowledge Work Plugins (11 role-based). Context editing & compaction.

10.3 Quick Reference ‚Äî Code Cheat Sheet

OpenAI

Simple call (Chat Completions)

openai.chat.completions.create(
  model="gpt-4o",
  messages=[{"role":"user","content":"Hi"}]
)

Simple call (Responses API)

openai.responses.create(
  model="gpt-4o",
  input="Hi"
)

With tools

openai.responses.create(
  model="gpt-4o",
  input="Search for...",
  tools=[{"type":"web_search"}]
)

Multi-turn (stateful)

r2 = openai.responses.create(
  model="gpt-4o",
  input="Follow up...",
  previous_response_id=r1.id
)

Anthropic

Simple call

anthropic.messages.create(
  model="claude-sonnet-4-5",
  max_tokens=1024,
  messages=[{"role":"user","content":"Hi"}]
)

With system prompt

anthropic.messages.create(
  model="claude-sonnet-4-5",
  max_tokens=1024,
  system="You are a DNS expert.",
  messages=[{"role":"user","content":"Hi"}]
)

With tools

anthropic.messages.create(
  model="claude-sonnet-4-5",
  max_tokens=1024,
  messages=[...],
  tools=[{"name":"get_weather",
          "input_schema":{...}}]
)

Multi-turn (manual)

messages.append(
  {"role":"assistant","content":r1.content})
messages.append(
  {"role":"user","content":"Follow up..."})
r2 = anthropic.messages.create(
  model="claude-sonnet-4-5",
  max_tokens=1024,
  messages=messages
)
